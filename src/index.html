Sobre o curso:

Curso tecnólogo da PUC Goiás com duração de 3 anos, ofertado no período noturno nas modalidades presencial e EaD. Forma profissionais capacitados para criar, manter e gerenciar sistemas de software, desenvolver aplicativos para computadores e dispositivos móveis, administrar bancos de dados e atuar com metodologias ágeis — atendendo à alta demanda do mercado de TI em todos os setores da economia.

Disciplinas

Laboratório de Programação em java

aplicação prática de algoritmos e estruturas de dados. O conteúdo envolve o estudo da linguagem Java

Engenharia de Software

capacita o aluno a planejar, especificar e documentar sistemas

Fundamentos de Computação I

introdução ao pensamento lógico, essencial para a programação. O conteúdo abrange lógica proposicional, estruturas matemáticas discretas

Algoritmos

ensina a criar sequências lógicas passo a passo para resolver problemas, usando ferramentas como pseudocódigo e fluxogramas

Introdução a Estatística para Inteligência Artificial

fundamentos matemáticos, algoritmos de Machine Learning, compreender como a IA quantifica incertezas e toma decisões baseadas em dados.

Fundamentos de Programação Orientada a Objetos

ensina a modelar sistemas do mundo real como códigos. O foco é trabalhar com Classes, Objetos e os 4 Pilares da POO.

Engenharia de Requisitos

foca em descobrir, analisar, especificar, validar e gerenciar as necessidades de um sistema

Projeto de Banco de Dados

transformação de regras de negócio em modelos de dados eficientes, englobando modelagem conceitual, lógica, física e a implementação em banco de dados relacional

Desenvolvimento de Software WEB

foca na construção prática de aplicações, aborda desde a estruturação de interfaces (HTML/CSS) até o consumo de serviços, usabilidade, testes e design.

Mensageria e Streams em aplicações

ensina a arquitetura de sistemas distribuídos focada na comunicação assíncrona

Design de Software

arquitetura limpa e princípios de orientação a objetos para criar sistemas robustos, modulares, fáceis de manter e de escalar

Modelagem de Interfaces de Usuário

estética, psicologia cognitiva e programação para criar aplicações intuitivas, cobrindo desde a teoria de UX/UI até a criação de protótipos
