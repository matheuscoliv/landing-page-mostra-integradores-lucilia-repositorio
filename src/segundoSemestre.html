Sobre o Projeto do Segundo Semestre:
TechFood

Gerador de planos alimentares personalizados de acordo com a idade e comorbidade de cada usuário. O sistema fazia uma requisição via API do Google genAI e com base na entrada que o usuário fornecia era retornado um plano alimentar personalizado. Ademais, persistência de dados dos usuários, planos alimentares, vinculos... são armazenados no banco de dados SQL.
